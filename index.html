What is Css

  1. CSS stands for Cascading Style Sheets.

  2. CSS describe how HTML elements are to be displayed on screen ,paper,or in other media.

  3. Css is not necessery CSS saves a lot of work. It can control the layout of the multiple web pages all at once

  4. External stylesheet are shorted in CSS files.

CSS can be add to HTML document in 3 ways:

  • inline-by using the style attribute inside HTML elemdentsl
  • internal-by using a style element in the head section
  • external-using a link element to link to an external CSS file
  • The most common way to add CSS ,is to Keep the style in external CSS files.However,in this tutorial we will use inline and internal styles,because this is easier to demonstrate,and easier for you to try it yourself.

when comparing CSS class vs ID, the difffrence is that CSS class applies a style to multtiple element.ID, on the other hand,applies a style to one unique element.ID is also special in that you can use a special URL to link directly to an element and its used by JavaScript.

in CSS, selectors are used to target a specific element or range of elements on a webpage.Once an element has been targeted,a style or set of style can be applied to the element.

There is wide range of selector available.Two of the commonly used are class and ID. Both are use to target element to which style should be applied.
